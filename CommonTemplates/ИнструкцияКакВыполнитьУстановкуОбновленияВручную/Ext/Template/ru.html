Установка обновления конфигурации

Для установки обновления конфигурации необходимо выполнить следующие действия:

  1. Если с программой работают другие пользователи, рекомендуется для установки обновления конфигурации выбрать удобное время, когда с программой никто не работает.
  2. Завершить работу всех пользователей.
  3. Завершить свой сеанс работы в программе.
  4. Запустить 1С:Предприятие в режиме Конфигуратор.
  5. В Конфигураторе в меню Конфигурация выбрать пункт Обновить конфигурацию базы данных (либо нажать на F7).
  6. Закрыть Конфигуратор.
  7. Запустить 1С:Предприятие и продолжить работу в программе.
