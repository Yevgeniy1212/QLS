Предназначена для настройки синхронизации файлов для каждого владельца (документа, папки и других объектов программы).

Открывается из списка Присоединенные файлы объектов программы по команде Еще - Синхронизация с облаком.

Настройка синхронизации файлов

  • Включите флажок Синхронизировать файлы с облачным сервисом.
  • Выберите из списка учетную запись облачного сервиса, с которым будет производиться синхронизация файлов. Если в списке нет нужной учетной записи, ее можно создать.
  • Для подтверждения нажмите Записать и закрыть.
