Предназначена для сохранения текущего варианта отчета в архивный файл вместе с пользовательскими настройками.

Открывается из окна варианта отчета с помощью команды Еще - Варианты отчета - Сохранить в файл. Для администратора в списке всех отчетов предусмотрена кнопка  Сохранить в файл. 

Укажите путь к архивному файлу на компьютере. Вместе с вариантом отчета по умолчанию сохраняются и пользовательские настройки. Можно отменить сохранение всех или некоторых настроек с помощью флажков.

Данная возможность предназначена для обмена настройками вариантов отчетов между информационными базами идентичной структуры.
