Открывается из варианта отчета по команде Еще - Изменить условия отборов, а также из настроек варианта отчета с помощью кнопки .

В списке выводится:

  • Поле - поле отчета, по которому предусмотрен отбор;
  • Условие - выводится вид сравнения по данному полю.

Изменение условия

  • С помощью двойного щелчка мыши по условию открывается список, из которого можно выбрать новое условие для отбора по соответствующему полю.
