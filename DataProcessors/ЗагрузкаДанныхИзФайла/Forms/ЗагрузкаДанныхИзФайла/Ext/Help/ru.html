Загрузка данных из файла в списки и документы

Открывается из документов и элементов списков с помощью кнопки Загрузить из файла.

С помощью команды Загрузка данных из файла для администратора имеется возможность загрузки данных в произвольный список программы.

С помощью кнопки выберите, как будут загружаться данные:

  • заполнение таблицы в программе;
  • загрузка из внешнего файла.

Заполнение таблицы

  • С помощью кнопки Заполнение таблицы (включена по умолчанию) можно перейти к вводу данных в таблицу для последующей загрузки в документ непосредственно в программе.
  • Таблицу можно заполнить, скопировав данные в таблицу из внешнего файла через буфер обмена.
  • В таблице имеются обязательные для заполнения колонки. Они перечисляются над таблицей.
  • С помощью ссылки Изменить бланк можно открыть список полей таблицы для ввода и с помощью флажков изменить ее структуру. Необходимо учитывать, что поля, выделенные в таблице жирным шрифтом, являются обязательными для заполнения и трогать их нельзя.
    • Укажите с помощью флажков необходимые колонки. Можно настроить наименование колонок, их отображение и порядок следования для упрощения загрузки.
  • После заполнения таблицы для перехода к сопоставлению данных нажмите Далее.

Загрузка из внешнего файла

  • Нажав кнопку Из внешнего файла, можно загрузить данные из ранее подготовленного файла, сохраненного на компьютере, или сохранить бланк для заполнения и заполнить данные позже.
  • С помощью кнопки Сохранить бланк для заполнения можно сохранить таблицу для ввода данных в файле выбранного формата. Необходимо учитывать, что в данном файле имеются колонки, обязательные для заполнения. Укажите путь к файлу на компьютере. Программа присваивает имя файлу по умолчанию.
  • Можно заполнить внешние файлы для загрузки в документы. Заполните файл по рекомендованным правилам (обязательные для заполнения поля выделены в таблице жирным шрифтом).
  • С помощью кнопки Загрузить таблицу из файла можно загрузить данные из внешнего файла. Выберите путь к файлу на компьютере.

Сопоставление данных

  • После загрузки данных программа производит автоматическое сопоставление данных. Если автоматическое сопоставление не состоялось, то данные выводятся для сопоставления вручную.
  • Для того чтобы просмотреть сопоставленные или несопоставленные данные, можно использовать отбор с помощью поля Показывать. Несопоставленные данные помечаются красным. 
  • С помощью соответствующей кнопки можно Разрешить неоднозначность. Для этого в таблице выводятся совпадающие элементы. Необходимо выбрать один из них для загрузки.
  • Для того чтобы вручную сопоставить данные, откройте соответствующий список программы, найдите в нем подходящее значение или введите новое, затем нажмите Выбрать.
  • Кроме обязательных полей, сопоставляться могут также и необязательные. При загрузке поле можно сопоставить и включить в документ с помощью кнопки Выбрать из списка. Если нужного элемента в списке нет, то его можно Создать с помощью соответствующей кнопки.
  • После автоматического или ручного сопоставления данных, в случае, если все данные сопоставлены, нажмите кнопку Загрузить данные. После этого данные будут загружены в документ или список. Остается лишь заполнить недостающие поля.

Загрузка данных в произвольный список программы (только для администратора)

  • Загрузку данных из файла в произвольный список администратору можно произвести по соответствующей команде.
  • В этом случае для загрузки данных можно выбрать один из списков программы. Можно загружать в списки контактную информацию, а также дополнительные реквизиты и сведения.
    • После выбора списка для загрузки данных нажмите кнопку Далее. Выберите с помощью кнопок, как будут загружаться данные:
      • Программа предлагает таблицу для заполнения.
      • Также можно загрузить данные из внешнего файла.

Сопоставление данных (только для администратора)

  • Заполнив таблицу или скачав данные из файла, нажмите кнопку Далее. Необходимо провести сопоставление данных.
  • Поведение программы при сопоставлении данных можно настроить:
    • С помощью переключателя выбрать, как поступать с Сопоставленными элементами: Обновлять или Пропускать (по умолчанию).
    • С помощью переключателя выбрать, как поступать с Несопоставленными элементами: Создавать (по умолчанию) или Пропускать.
  • Данные будут автоматически сопоставлены, если соответствия не будет найдено, таблица выводится с предложением сопоставить данные.
    • В некоторых списках существует возможность автоматического создания несопоставленных данных. Такие данные помечаются надписью <Создан>. Для автоматического создания в списке выберите Создавать с помощью переключателя (выбрано по умолчанию).
    • После автоматического или ручного сопоставления данных, в случае, если все данные сопоставлены, нажмите кнопку Загрузить данные.
  • После этого данные будут загружены в документ или список. Остается лишь заполнить недостающие поля. Результаты загрузки выводятся на просмотр.
    • С помощью поля Показывать можно вывести Все, Новые, Обновленные и Пропущенные элементы.
    • Несколько выделенных строк можно Сохранить в файл, подкорректировать и позднее загрузить повторно.
    • В списке можно изменить введенные элементы, если они не устраивают. Для изменения нескольких выделенных элементов нажмите кнопку Изменить реквизиты.
    • При необходимости включите флажок Открыть справочник по кнопке "Готово".
  • Если в таблице все данные корректны, нажмите кнопку Готово для подтверждения. Если был включен флажок, открывается список с введенными элементами.
